We are testing for differential expression using both a negative binomial genarlized linear model and a quasipoisson generalized linear model. Each model will take in the TPM values for each transcript individually, and determine if the cell type predicts expression level.

We used two different machine learning techniques to test for differential expression in questions 1, 2 and 3. The first technique we used was the negative binomial generalized linear model (nb-glm). This was done using the R package edgeR. There are several reasons we use a nb-glm. One reason is because our transcript concentrations are count based. Simply put, our concentration was determined from the count of RNA fragments that overlapped a transcript. Since our concentration is count based, it cannot be normally distributed (it is impossible to have a negative amount of fragments, or a fraction of a fragment). Another reason is because we have a low number of samples (only 2 stem cells and two brain cells). This results in overdispersion, a common issue when analyzing genomic data. The negative binomial distribution does not require the estimated variance to be equal to the mean. So, overdispersion does not hinder the test for significant changes in transcript concentration. The second technique we used in questions 1, 2, and 3 was a quasipoisson glm (qp-glm). This was implemented with our data using a for loop. Similar to the nb-glm, qp-glm works with count data as well as overdispersion, making this a practical choice. The qp-glm was used to test if the cell type predicted the concentration of each transcript. A model was generated for each transcript and assigned a p-value based on the correlation between cell type and concentration. This p-value was used to determine if the transcript was differentially expressed. In summary, the variance and mean are not linked, and the data does not have to be normally distributed when using a qp-glm or nb-glm. This makes edgeR and quasipoisson a practical choice for ML model.
We came to the conclusion of using the nb-glm model from the edgeR package because after reading the approach others took in the community to answer similar questions, and inspecting the tools published for genomic expression data analyses, we found that this application is the most reliable and practical for our purposes. Although we were unable to find other approaches using a quasipoisson model, we decided to use it because of its ability to handle count based data as well as overdispersion.

Now that we have used a negative binomial model to predict differential expression, we will repeat the experiment using a quasipoisson model.

we start by making a vector that will hold our results

de_pois = vector()

We use a for loop to iterate through each transcrtipt and generate a model to show if expression level predicts cell type

for (row in 1:nrow(expr_data)) {
  
  ### grab each set of expression data per transcript ###
  tran <- expr_data[row,]
  tran <- (as.data.frame(t(as.matrix(tran)))) # transpose our data
  colnames(tran)[1] <- "expression"
  tran$cell_type <- c(1,1,0,0) # add a column for cell type (1=brain, 0=stem)
  
  ### Ignore cases where TPM is all zeros ###
  check <- list()
  for (i in 1:nrow(tran)) {
    if (tran[i, 1] != 0) {
      check <- append(check, 1)
    }
  }
  
  ### Make a logistic model to test if expression level predicts cell type
  if (length(check) > 0) {
    model <- glm(expression ~ cell_type, data = tran, family = quasipoisson)
  }
  
  ### save the p value, if less than .05 than cell type predicts expression ### 
  pVal <- (as.matrix(summary(model)$coefficients)[2,4])
  if (as.numeric(pVal) < .05){
    de_pois <- append(de_pois, rownames(expr_data[row,]))
  }
}

We again visualize our results with a heatmap. There were 88 DE transcripts detected with the qp-glm. Surprisingly, the qp-glm was more accurate than the nb-glm used by edgeR. This is shown with the consistent tile coloring between the brain cells and stem cells. Our qp-glm detected the same transcrpts as qp-glm, but substantially less.

Based on our results we can conclude that there are unknown, noncoding DE transcripts in our data and that a qp-glm is a better prediction model for DE transcripts than nb-glm in this case.
